# dotfiles

Clone this repo, install submodules, run `install` to set up.

This dotfiles uses [Dotbot](https://github.com/anishathalye/dotbot/) for installation.

## Fonts

- wucon siji
- misc ohsnap
- iosevka term
- powerline fonts
- dejavu sans mono
